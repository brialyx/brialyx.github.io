Obsolète

WEP (Wired Equivalent Privacy)

WEP = première sécurité Wi-Fi (1997). TOTALEMENT CASSÉ, ne JAMAIS utiliser.

// Faiblesses WEP

  • ❌ Chiffrement RC4 faible (40/104 bits)
  • ❌ IV (Initialization Vector) court 24 bits → réutilisé rapidement
  • ❌ Pas d'intégrité des données
  • ❌ Clé statique partagée (jamais changée)
  • ❌ Cassable en <5 minutes avec aircrack-ng
🚫
WEP = INTERDIT WEP cassé depuis 2001. Si tu vois WEP sur réseau production → ALERTE SÉCURITÉ CRITIQUE. Migrer immédiatement vers WPA2/WPA3.
Standards modernes

WPA/WPA2 — Personal vs Enterprise

WPA et WPA2 ont 2 modes d'authentification :

// WPA/WPA2-Personal (PSK)

PSK (Pre-Shared Key) = clé partagée (mot de passe Wi-Fi).

Caractéristiques :

  • Authentification : Mot de passe unique partagé par tous
  • Clé : 8-63 caractères ASCII
  • Usage : Maisons, SOHO, PME (simple)
  • Avantage : Facile à configurer
  • Inconvénient : Mot de passe partagé = faible sécurité

// WPA/WPA2-Enterprise (802.1X)

802.1X = authentification centralisée via serveur RADIUS.

Caractéristiques :

  • Authentification : Nom d'utilisateur + mot de passe OU certificat
  • Serveur RADIUS requis
  • Chaque utilisateur = identifiants uniques
  • Usage : Entreprises, universités, campus
  • Avantage : Sécurité forte, audit, révocation individuelle
  • Inconvénient : Complexe (infrastructure RADIUS)

// Comparaison Personal vs Enterprise

Standard actuel

WPA2 — AES-CCMP

WPA2 (2004) = standard actuel recommandé. Chiffrement AES-CCMP.

// WPA2 Améliorations

  • ✅ AES-CCMP (AES Counter Mode with CBC-MAC Protocol)
  • ✅ Chiffrement 128 bits (fort)
  • ✅ Intégrité données (MIC - Message Integrity Check)
  • ✅ Protection replay attacks
  • ✅ Compatible 802.11i (norme IEEE complète)

// TKIP vs AES-CCMP

CritèreTKIP (WPA)AES-CCMP (WPA2)
AlgorithmeRC4 amélioréAES (Advanced Encryption Standard)
Clé128 bits128 bits
Sécurité⚠️ Faible (attaques connues)✅ Forte
Performances⚠️ Lentes✅ Rapides (accélération HW)
Status❌ Obsolète✅ Standard
✅
Recommandation Toujours utiliser WPA2 avec AES-CCMP (pas TKIP). WPA2-Personal (PSK) pour SOHO. WPA2-Enterprise (802.1X) pour entreprises.
Nouveau standard

WPA3 — SAE et améliorations

WPA3 (2018) = nouveau standard. Remplace PSK par SAE.

// WPA3 Nouveautés

  • ✅ SAE (Simultaneous Authentication of Equals) → remplace 4-way handshake PSK
  • ✅ Protection attaques dictionnaire offline
  • ✅ Forward Secrecy : Clés session uniques, compromise clé ≠ décryptage ancien trafic
  • ✅ 192-bit security (WPA3-Enterprise)
  • ✅ Wi-Fi Easy Connect (QR codes pour IoT)
  • ✅ Enhanced Open : Chiffrement réseaux ouverts (OWE - Opportunistic Wireless Encryption)

// WPA2 vs WPA3

CritèreWPA2WPA3
Handshake4-way handshake (PSK)SAE (Dragonfly)
Attaque dictionnaire⚠️ Vulnérable offline✅ Protégé
Forward Secrecy❌ Non✅ Oui
Réseaux ouverts❌ Pas chiffré✅ Enhanced Open (OWE)
Adoption✅ Large⚠️ Croissante
🚀
Migrer vers WPA3 WPA3 recommandé nouveaux déploiements 2024+. Équipements récents supportent. Si pas compatible WPA3 → WPA2 toujours acceptable.
Authentification Enterprise

EAP (Extensible Authentication Protocol)

EAP = framework d'authentification pour 802.1X (WPA/WPA2/WPA3-Enterprise).

// Types EAP principaux

1. EAP-TLS (Transport Layer Security)

  • Authentification mutuelle par certificats (client + serveur)
  • ✅ Très sécurisé
  • ⚠️ Complexe (déploiement certificats)
  • Usage : Environnements haute sécurité

2. PEAP (Protected EAP)

  • Tunnel TLS chiffré → username/password à l'intérieur
  • Certificat serveur uniquement (pas client)
  • ✅ Sécurisé + simple
  • Usage : Standard entreprises (Microsoft, Cisco)
  • Sous-types : PEAP-MSCHAPv2, PEAP-GTC

3. EAP-TTLS (Tunneled TLS)

  • Similaire PEAP (tunnel TLS)
  • Support plus de méthodes authentification internes
  • Usage : Alternative PEAP (moins répandu)

4. EAP-FAST (Flexible Authentication via Secure Tunneling)

  • Cisco propriétaire
  • Pas besoin certificats (PAC - Protected Access Credential)
  • Usage : Environnements Cisco

// Comparaison EAP

Type EAPCertificat clientCertificat serveurSécuritéComplexité
EAP-TLS✅ Oui✅ Oui✅ Très haute⚠️ Élevée
PEAP❌ Non✅ Oui✅ Haute✅ Moyenne
