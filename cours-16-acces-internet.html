Ligne téléphonique

DSL — Digital Subscriber Line

DSL = Internet sur ligne téléphonique cuivre existante. Fréquences séparées voix/data → téléphone + Internet simultanés.

// ADSL (Asymmetric DSL)

Asymétrique : Download >> Upload

  • Download : 1-24 Mbps
  • Upload : 0.5-3 Mbps
  • Distance max : ~5 km du central téléphonique (DSLAM)
  • ✅ Bon pour navigation web, streaming
  • ⚠️ Upload faible (problème visioconférence, cloud backup)

// VDSL / VDSL2 (Very-high-bit-rate DSL)

DSL haute vitesse : Version améliorée ADSL

  • Download : 50-100 Mbps
  • Upload : 10-40 Mbps
  • Distance max : ~1.5 km (plus court que ADSL)
  • ✅ Upload décent (visio OK)
  • ⚠️ Portée limitée

// Problème DSL : Distance

Débit DSL dégrade avec distance du central (DSLAM). Plus loin = plus lent.

📏
Atténuation signal DSL Maison à 4 km du central → ADSL 5 Mbps au lieu de 24 Mbps annoncés.
Câble/Fibre pas affectés par distance (partagé différemment).
Câble coaxial

Câble — DOCSIS

Câble = Internet via réseau TV câble (coaxial). Standard DOCSIS (Data Over Cable Service Interface Specification).

// Versions DOCSIS

VersionDownload maxUpload maxAnnée
DOCSIS 3.0~1 Gbps~200 Mbps2006
DOCSIS 3.1~10 Gbps~2 Gbps2013
DOCSIS 4.0~10 Gbps~6 Gbps2017 (déploiement progressif)

// Caractéristiques

  • ✅ Débits élevés (100-1000 Mbps courant)
  • ✅ Latence faible (~15ms)
  • ✅ Infrastructure existante (TV câble)
  • ⚠️ Bande passante partagée : Dégradation heures pointe (voisinage)
  • ⚠️ Upload asymétrique (moins que download)

// Topologie

Réseau câble = architecture partagée (arbre).

[Head-end ISP] → [Nœud quartier] → Split 32-500 foyers
↓ (partagé)
[Cable modem maison 1]
[Cable modem maison 2]
...
[Cable modem maison 500]

→ Tous foyers partagent même pipe
→ Heures pointe (19h-22h) = congestion possible
🎯
Question type examen "Utilisateur se plaint débit Internet lent le soir uniquement. Techno probable ?"

Réponse : Câble (bande passante partagée quartier → congestion heures pointe)
Fibre optique

Fibre — GPON / FTTH

Fibre = Internet par fibre optique. FTTH (Fiber-To-The-Home) = fibre jusqu'au domicile.

// GPON (Gigabit Passive Optical Network)

Standard fibre résidentiel dominant.

  • Download/Upload : Symétriques (100/100, 1000/1000, 10000/10000 Mbps)
  • Latence : ~5ms (ultra-faible)
  • Portée : ~20 km
  • Architecture PON : 1 fibre split vers 32-128 abonnés

// Variantes déploiement fibre

FTTH (Fiber-To-The-Home) : Fibre jusqu'à la maison (meilleur)

FTTC (Fiber-To-The-Curb) : Fibre jusqu'au trottoir → dernier segment cuivre/DSL

FTTN (Fiber-To-The-Node) : Fibre jusqu'au nœud quartier → dernier km cuivre

// Avantages fibre

  • ✅ Débits ultra-élevés (multi-Gbps)
  • ✅ Symétrique (upload = download)
  • ✅ Latence ultra-faible (~5ms)
  • ✅ Pas d'atténuation distance
  • ✅ Pas d'interférences électromagnétiques
  • ⚠️ Coût installation (si pas infrastructure)
🚀
Fibre = gold standard 2024 Meilleure technologie accès Internet résidentiel/PME. Déploiements massifs en cours mondialement. Symétrique, ultra-rapide, faible latence.
Sans fil spatial

Satellite — accès zones isolées

Satellite = Internet via antenne parabolique communicant avec satellite géostationnaire/LEO.

// Types satellites

Géostationnaires (GEO) : ~35,786 km altitude

  • Latence : ~500-700ms (aller-retour 70,000+ km)
  • Débits : 25-100 Mbps
  • Exemples : HughesNet, Viasat

LEO (Low Earth Orbit) : ~550 km altitude

  • Latence : ~20-40ms (bien meilleur !)
  • Débits : 50-200 Mbps
  • Exemples : Starlink (SpaceX), OneWeb

// Avantages/Inconvénients

  • ✅ Disponible partout (zones rurales isolées)
  • ⚠️ Latence élevée (GEO = inutilisable gaming/VoIP)
  • ⚠️ Coût élevé (équipement + abonnement)
  • ⚠️ Météo : Pluie/neige dégradent signal
  • ⚠️ Caps data souvent (limites mensuelles)
🛰️
Starlink change la donne Constellation LEO Starlink = latence ~20-40ms (vs 500ms GEO) → gaming/visio possibles. Révolution accès rural.
Mobile broadband

Cellulaire — 4G LTE / 5G NR

// 4G LTE (Long-Term Evolution)

4ème génération réseau mobile.

  • Download : 10-100 Mbps (théorique 300 Mbps)
  • Upload : 5-30 Mbps
  • Latence : ~30ms
  • ✅ Large couverture
  • ⚠️ Débits variables (congestion, distance antenne)

// 5G NR (New Radio)

5ème génération — révolution.

  • Download : 100-3000 Mbps (théorique 10+ Gbps)
  • Upload : 50-500 Mbps
  • Latence : ~10ms (ultra-faible)
  • ✅ Débits ultra-élevés
  • ✅ Latence faible (gaming, VR, IoT)
  • ⚠️ Couverture limitée (déploiement progressif)
  • ⚠️ Portée courte (hautes fréquences mmWave)

// Usage accès Internet fixe (FWA)

FWA (Fixed Wireless Access) : Box 4G/5G comme accès Internet principal maison/bureau (pas juste mobile).

  • Alternative zones sans fibre/câble
  • Installation rapide (pas travaux)
  • ⚠️ Caps data souvent
⚠️ Pièges d'examen

FAUX. Câble = bande passante partagée quartier.

Congestion heures pointe possible.
Fibre GPON aussi partagée MAIS capacité bien supérieure

FAUX pour GEO. Satellite géostationnaire = ~500ms latence.

Starlink (LEO) = ~20-40ms (acceptable)
DSL/Câble/Fibre = ~5-20ms
Résumé

🔑 Ce que tu dois retenir absolument

→DSL : Ligne téléphonique, asymétrique, débit dégrade avec distance. ADSL (24 Mbps), VDSL (100 Mbps)
→Câble (DOCSIS) : Coaxial, partagé quartier, 100-1000 Mbps, congestion heures pointe possible
→Fibre (GPON) : Symétrique, ultra-rapide (multi-Gbps), latence ~5ms, gold standard 2024
→Satellite : Zones isolées. GEO (500ms latence), LEO/Starlink (20-40ms latence acceptable)
→Cellulaire : 4G LTE (100 Mbps, 30ms), 5G NR (3 Gbps, 10ms). FWA = accès fixe via mobile
