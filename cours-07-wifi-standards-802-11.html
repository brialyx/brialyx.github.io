Avant de commencer

Standards Wi-Fi — évolution des normes 802.11

Wi-Fi = nom commercial des normes IEEE 802.11 (réseaux sans fil). Wi-Fi Alliance certifie compatibilité entre équipements.

IEEE 802.11 = famille de standards définissant communication sans fil WLAN. Chaque lettre (a/b/g/n/ac/ax) = révision avec améliorations.

Pour l'examen Network+ CompTIA teste : débits théoriques (54Mbps, 600Mbps, 1.3Gbps...), bandes fréquences (2.4GHz vs 5GHz vs 6GHz), compatibilités, MIMO, Wi-Fi 6/6E. Mémoriser tableau comparatif !

Fréquences

2.4 GHz vs 5 GHz vs 6 GHz

// Bande 2.4 GHz

Standards : 802.11b, 802.11g, 802.11n, 802.11ax

Avantages :

  • ✅ Portée longue (traverse mieux les murs)
  • ✅ Compatible anciens équipements
  • ✅ Meilleure pénétration obstacles

Inconvénients :

  • ❌ Seulement 3 canaux non-chevauchants (1, 6, 11)
  • ❌ Très encombrée (Bluetooth, micro-ondes, baby monitors...)
  • ❌ Débits plus faibles
  • ❌ Interférences fréquentes

// Bande 5 GHz

Standards : 802.11a, 802.11n, 802.11ac, 802.11ax

Avantages :

  • ✅ 23+ canaux non-chevauchants (moins d'interférences)
  • ✅ Débits plus élevés
  • ✅ Moins encombrée
  • ✅ Meilleure pour streaming/gaming

Inconvénients :

  • ⚠️ Portée plus courte
  • ⚠️ Traverse moins bien les obstacles
  • ⚠️ Incompatible anciens équipements (802.11b/g uniquement 2.4GHz)

// Bande 6 GHz (Wi-Fi 6E)

Standards : 802.11ax uniquement

Avantages :

  • ✅ Spectre vierge (pas d'équipements legacy)
  • ✅ Énormément de canaux (~59 canaux de 20MHz)
  • ✅ Zéro interférences
  • ✅ Latence ultra-faible

Inconvénients :

  • ⚠️ Requiert équipements récents (Wi-Fi 6E)
  • ⚠️ Portée encore plus courte que 5GHz
  • ⚠️ Pas encore très répandu
💡
Conseil pratique 2.4 GHz : IoT, appareils lointains, compatibilité legacy
5 GHz : Laptops, smartphones, streaming, gaming (standard actuel)
6 GHz : Applications critiques latence, VR/AR, futurs usages
Détails standards

802.11n (Wi-Fi 4)

Année : 2009
Bandes : 2.4 GHz ET 5 GHz (dual-band)
Débit max : 600 Mbps (théorique, 4 flux MIMO)

// Nouveautés 802.11n

  • MIMO (Multiple Input Multiple Output) : Jusqu'à 4 antennes TX/RX simultanées
  • Channel bonding : 2 canaux de 20MHz fusionnés = 40MHz (double débit)
  • Frame aggregation : Plusieurs trames en 1 transmission (efficacité)
  • Backward compatible : Compatible 802.11a/b/g

Configuration typique : 2x2:2 MIMO (2 TX, 2 RX, 2 flux spatiaux) → ~300 Mbps

Détails standards

802.11ac (Wi-Fi 5)

Année : 2014
Bandes : 5 GHz uniquement
Débit max : 3.5 Gbps (Wave 2, 8 flux MIMO)

// Nouveautés 802.11ac

  • Canaux 80MHz et 160MHz : Channel bonding étendu
  • MU-MIMO (Multi-User MIMO) : AP communique avec plusieurs clients simultanément (Wave 2)
  • Modulation 256-QAM : Plus de données par symbole
  • Beamforming : Signal dirigé vers client (meilleure portée/débit)
  • 8 flux spatiaux max (vs 4 en 802.11n)

Wave 1 vs Wave 2 :

  • Wave 1 (2014) : Jusqu'à 3.5 Gbps, pas de MU-MIMO
  • Wave 2 (2016) : MU-MIMO, 160MHz channels, 8 flux
⚠️
802.11ac = 5GHz uniquement Pas compatible 2.4GHz. Si client 2.4GHz uniquement → fallback 802.11n.
Détails standards

802.11ax (Wi-Fi 6 / Wi-Fi 6E)

Année : 2019
Bandes : 2.4 GHz, 5 GHz, 6 GHz (6E)
Débit max : 9.6 Gbps (théorique)

// Nouveautés Wi-Fi 6

  • OFDMA (Orthogonal Frequency Division Multiple Access) : Subdivision canaux, plusieurs clients simultanés sur même canal
  • MU-MIMO amélioré : Uplink ET downlink (802.11ac = downlink uniquement)
  • 1024-QAM : Modulation plus dense (+25% débit vs 802.11ac)
  • TWT (Target Wake Time) : Économie batterie appareils IoT
  • BSS Coloring : Réduction interférences réseaux voisins
  • Meilleur en environnements denses : Stades, aéroports, campus

// Wi-Fi 6 vs Wi-Fi 6E

Technologie

MIMO et MU-MIMO

// MIMO (Multiple Input Multiple Output)

Plusieurs antennes TX (transmission) et RX (réception) pour envoyer/recevoir plusieurs flux de données simultanément.

Notation : 3x3:2 = 3 antennes TX, 3 antennes RX, 2 flux spatiaux (streams)

// SU-MIMO (Single-User MIMO)

AP communique avec 1 seul client à la fois, mais utilise plusieurs flux spatiaux.

  • 802.11n et 802.11ac Wave 1 = SU-MIMO uniquement
  • AP avec 4 antennes peut envoyer 4 flux vers 1 client
  • Autres clients attendent leur tour

// MU-MIMO (Multi-User MIMO)

AP communique avec plusieurs clients simultanément.

  • Introduit en 802.11ac Wave 2 (downlink uniquement)
  • 802.11ax (Wi-Fi 6) : Downlink ET uplink
  • AP avec 8 antennes peut servir 4 clients en même temps (2 flux chacun)
  • Améliore efficacité en environnements denses
🎯
Question type examen "Quel standard Wi-Fi introduit MU-MIMO ?"

Réponse : 802.11ac Wave 2 (downlink)
802.11ax améliore avec uplink + downlink
⚠️ Pièges d'examen

Ce qui trompe les candidats

❌ Piège #1 : 802.11ac = 2.4 + 5 GHz
FAUX. 802.11ac = 5 GHz UNIQUEMENT.

802.11n et 802.11ax = dual-band (2.4 + 5 GHz)
802.11ax (6E) = tri-band (2.4 + 5 + 6 GHz)
❌ Piège #2 : Wi-Fi 6 = 802.11ac
FAUX.
Wi-Fi 6 = 802.11ax
Wi-Fi 5 = 802.11ac
Wi-Fi 4 = 802.11n
❌ Piège #3 : MU-MIMO introduit en 802.11n
FAUX. MU-MIMO introduit en 802.11ac Wave 2.

802.11n = MIMO (SU-MIMO)
802.11ac Wave 2 = MU-MIMO downlink
802.11ax = MU-MIMO downlink + uplink
❌ Piège #4 : 802.11b = 54 Mbps
FAUX. 802.11b = 11 Mbps.

54 Mbps = 802.11a ET 802.11g
