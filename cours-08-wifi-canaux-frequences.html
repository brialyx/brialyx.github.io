Bande 2.4 GHz

Canaux 2.4 GHz — le problème du chevauchement

🎵
Analogie — stations radio Imagine 3 stations radio : 88.1 FM, 88.3 FM, 88.5 FM. Si trop proches, elles se brouillent mutuellement. En Wi-Fi 2.4GHz, c'est pareil : canaux trop proches = interférences. Solution : espacer avec canaux 1, 6, 11 (assez loin pour ne pas se chevaucher).

Bande totale : 2.400 - 2.4835 GHz (83.5 MHz)
Canaux disponibles : 1 à 14 (selon pays)
Largeur canal : 20 MHz

// Canaux non-chevauchants : 1, 6, 11

Un canal Wi-Fi 20MHz occupe ±10 MHz autour de sa fréquence centrale → chevauchement des canaux voisins.

Solution : Utiliser canaux espacés d'au moins 5 numéros → canaux 1, 6, 11 = seuls 3 canaux non-chevauchants.

🎯
Question TRÈS fréquente à l'examen "Quels canaux 2.4 GHz utiliser pour éviter interférences ?"

Réponse : 1, 6, 11

// Interférences 2.4 GHz

Sources principales :

  • Micro-ondes (~2.45 GHz) — ennemi #1
  • Bluetooth (2.4 GHz)
  • Baby monitors (2.4 GHz)
  • Téléphones sans fil (2.4 GHz)
Bande 5 GHz

Canaux 5 GHz — beaucoup plus d'espace

23+ canaux non-chevauchants en 20MHz (vs 3 en 2.4GHz).

DFS (Dynamic Frequency Selection) = éviter interférences avec radars météo/militaires. AP change canal si radar détecté.

RSSI et SNR

// RSSI (Received Signal Strength Indicator)

Puissance signal reçu en dBm. Plus proche de 0 = meilleur.

RSSIQualité
-30 à -50 dBm✅ Excellent
-50 à -60 dBm✅ Très bon
-60 à -70 dBm⚠️ Bon
-70 à -80 dBm⚠️ Faible
< -90 dBm❌ Inutilisable

// SNR (Signal-to-Noise Ratio)

Rapport signal/bruit en dB. Plus élevé = meilleur.

Formule : SNR = Signal (dBm) - Bruit (dBm)

SNRQualité
> 40 dB✅ Excellent
25-40 dB✅ Très bon
15-25 dB⚠️ Bon
< 10 dB❌ Mauvais
⚠️ Pièges d'examen

Ce qui trompe les candidats

FAUX. Canaux non-chevauchants 2.4 GHz = 1, 6, 11
❌ Piège #2 : RSSI -30 dBm = mauvais
FAUX. RSSI -30 dBm = EXCELLENT. Plus proche de 0 = meilleur.
Résumé

🔑 Ce que tu dois retenir absolument

→Canaux non-chevauchants 2.4 GHz : 1, 6, 11

→Interférences 2.4GHz : micro-ondes (#1), Bluetooth, baby monitors
→RSSI : dBm, plus proche 0 = meilleur (-30 excellent, -90 mauvais)
→SNR : dB, plus élevé = meilleur (>40 dB excellent, <10 dB mauvais)
